8.2.10.6 datetime.datestrs ¶

Method: out = datestrs (obj) ¶
Method: out = datestrs (obj, …) ¶

Format obj as date strings, returning cellstr. Supports all arguments that core Octave’s datestr does.

Returns a cellstr array the same size as obj.
